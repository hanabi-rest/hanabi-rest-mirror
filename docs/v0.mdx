[v0.dev](https://v0.dev) is a product operated by Vercel, capable of generating components using [Tailwind CSS](https://tailwindcss.com) and [shadcn/ui](https://ui.shadcn.com) through AI. With hanabi.rest, it's possible to automatically design APIs based on the components generated by v0.

## 1. Generating Components with v0

All components created and published through v0 can be utilized for API generation in hanabi.rest. As of now, private components are not available for use.
